# 徐大源手工冰棍会员管理小程序部署说明

## 1. 导入项目

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹：`徐大源手工冰棍小程序`。
4. 在 `project.config.json` 中把 `appid` 改为正式小程序 AppID。

## 2. 开通微信云开发

1. 在微信开发者工具顶部点击“云开发”。
2. 创建云环境，例如：`xudayuan-prod`。
3. 复制云环境 ID。
4. 打开 `app.js`，把 `cloudEnv` 改成你的云环境 ID。

## 3. 创建云数据库集合

在云开发控制台创建 3 个集合：

- `members`：会员资料、等级、生日、偏好、备注
- `member_notes`：到店记录、购买备注、经营备注
- `reminders`：客户跟进提醒

权限建议先使用“仅创建者可读写”，门店内部使用时可按账号角色继续收紧。

## 4. 上传并部署云函数

在微信开发者工具左侧找到 `cloudfunctions`：

1. 右键 `memberService`，选择“上传并部署：云端安装依赖”。
2. 右键 `consumeService`，选择“上传并部署：云端安装依赖”。
3. 右键 `reminderService`，选择“上传并部署：云端安装依赖”。
4. 右键 `seedDemo`，选择“上传并部署：云端安装依赖”。

## 5. 初始化演示数据

部署完成后打开小程序：

1. 进入“设置”。
2. 点击“初始化演示数据”。
3. 回到首页查看会员总览。

## 6. 下午演示话术

这是徐大源手工冰棍的微信云开发会员系统，不是临时网页：

- 会员资料保存在微信云数据库。
- 本系统不开通微信支付，不做在线收款。
- 会员资料和到店记录保存在微信云数据库。
- 每次记录可自动生成客户跟进文案。
- 店员可以复制通知，通过微信发给客户。
- 数据结构支持后续扩展短信通知、Excel 批量导入和角色权限。

## 7. 正式上线前还需要确认

- 小程序 AppID
- 微信云环境 ID
- 店员登录和权限范围
- 是否需要短信服务商自动发送客户跟进提醒
- 现有会员 Excel 数据字段
- GitHub 目标仓库地址
