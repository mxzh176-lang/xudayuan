# 徐大源手工冰棍会员管理小程序

微信云开发版会员信息管理系统，适用于手工冰棍门店记录会员资料、口味偏好、到店备注和客户跟进提醒。

## 功能

- 会员资料管理
- 会员资料记录
- 口味偏好与生日记录
- 到店/购买备注记录
- 客户跟进提醒
- 不开通微信支付，不做在线收款
- 微信云数据库存储
- 云函数统一校验业务规则
- 演示数据初始化

## 云数据库集合

- `members`
- `member_notes`
- `reminders`

## 云函数

- `memberService`：会员查询、保存、首页统计
- `consumeService`：会员到店记录、备注保存、提醒生成
- `reminderService`：提醒列表、标记已通知
- `seedDemo`：创建演示数据

## 快速开始

1. 用微信开发者工具导入本项目。
2. 修改 `project.config.json` 的 `appid`。
3. 修改 `app.js` 的 `cloudEnv`。
4. 创建云数据库集合。
5. 上传部署 4 个云函数。
6. 在小程序“设置”页初始化演示数据。

详细步骤见 `docs/微信云部署说明.md`。
