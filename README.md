# 徐大源手工冰棍会员管理小程序

微信云开发版会员管理系统，适用于手工冰棍门店管理会员、余额、积分、消费记录和客户提醒。

## 功能

- 会员资料管理
- 会员余额扣款
- 消费积分累计
- 消费后自动生成客户提醒
- 低余额会员提示
- 微信云数据库存储
- 云函数统一校验业务规则
- 演示数据初始化

## 云数据库集合

- `members`
- `consumption_records`
- `reminders`

## 云函数

- `memberService`：会员查询、保存、首页统计
- `consumeService`：消费扣款、积分累计、提醒生成
- `reminderService`：提醒列表、标记已通知
- `seedDemo`：创建演示数据

## 快速开始

1. 用微信开发者工具导入本项目。
2. 修改 `project.config.json` 的 `appid`。
3. 修改 `app.js` 的 `cloudEnv`。
4. 创建云数据库集合。
5. 上传部署 4 个云函数。
6. 在小程序“设置”页初始化演示数据。

详细步骤见 `docs/微信云部署说明.md`。
